Bienvenue

Bienvenue sur Jàng Afrig , un quiz géographique sur les 54 pays africains . Choisissez un mode de jeu et battez votre meilleur score !

Les modes de jeu

  • Capitales : « Quelle est la capitale du Mali ? »
  • Drapeaux : « À quel pays appartient ce drapeau ? »
  • Langues : « Dans quel pays parle -t-on le swahili ? »
  • Voisins : « Combien de voisins a l'Algérie ? »
  • Chrono : maximum de bonnes réponses en 60 secondes

À propos de Jàng Afrig

Jàng Afrig est une application web de quiz géographique sur les 54 pays africains . Le mot jàng signifie « apprendre , étudier » en wolof.

Ce projet est développé dans le cadre du cours Fondamentaux du Web en L1 DUT1-Informatique à l'École Supérieure Polytechnique (ESP/UCAD).

Sources des données

Les données sur les pays sont fournies par REST Countries v3.1, une API libre et gratuite.

Technologies utilisées

  • HTML5 pour la structure
  • CSS3 pour la mise en forme
  • JavaScript moderne pour l'interactivité
  • Git et GitHub pour le versioning et l'hébergement
